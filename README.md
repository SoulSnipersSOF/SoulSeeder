🫡 Welcome, Souldier! Thank you for downloading the SoulSeeder app! This is a custom app developed by me, SoulSniper, to help our community members out by automating the process of seeding our servers. 

---

⚙️ What It Does

✅ Launches Hell Let Loose automatically
✅ Queries real-time player counts from BattleMetrics
✅ Joins Soul 1 if it’s below the player threshold (e.g., 70 players)
✅ Monitors population while idle in the background
✅ Leaves Soul 1 once it’s full
✅ Joins Soul 2 if the player count is below 70
✅ Monitors population while idle in the background
✅ Stops once both servers are full

---

🧠 How to Use

1. Run the installer and follow the prompts  
2. That’s it. You’re done. Go make yourself a sandwich and then watch some SoulSniper videos on YouTube. 

If you ever want to start it manually, just launch the app from your desktop or Start Menu.
